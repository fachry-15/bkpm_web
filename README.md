
# Laravel

Repository untuk tugas mata kuliah workshop web framework


## Run Locally

Clone the project

```bash
  git clone https://github.com/Insectivy/laravel-app.git
```

Go to the project directory

```bash
  cd laravel-app
```

Install dependencies

```bash
  composer install
  npm install
  npm run dev
```

Setup environment

```bash
  cp .env.example .env
  php artisan key:generate
  php artisan migrate
```

Start the server

```bash
  php artisan serve
```
